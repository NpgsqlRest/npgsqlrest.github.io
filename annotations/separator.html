SEPARATOR ​

Set the column separator for raw output mode.

Keywords ​

separator, raw_separator

Syntax ​

separator <string>
1

Supports escape sequences: \t (tab), \n (newline), \\ (backslash)

Examples ​

Comma Separator (CSV) ​

sql
comment on function export_csv() is
'HTTP GET
